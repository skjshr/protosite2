# Excavation App

作業セッションを収集と成長に変換するWebアプリのMVP。

このプロジェクトは、勉強や作業の開始ハードルを下げつつ、
継続によって報酬、図鑑、レベルアップが進む仕組みを作ることを目的とする。

今の段階では、画像、演出、ドット絵は入れない。
まずは骨格、状態管理、スコア計算、データ保存を優先する。

## MVPの目的

- 作業を開始しやすくする
- 作業終了時に報酬が得られる
- 作業時間が累積し、成長が可視化される
- 長時間耐久より、効率の良い作業リズムを褒める
- 将来的なデスクトップアプリ化に耐えられる構造で作る

## 現在のMVP範囲

- ログイン
- 用途フィールドの作成
- 用途フィールド一覧
- セッション開始
- 休憩
- 再開
- 終了
- セッション結果表示
- 経験値の加算
- レベルアップ
- 用途ごとの累計時間表示
- 図鑑表示
- 近傍ランキング表示

## 今はやらないこと

- ドット絵
- アニメーション演出
- 通知の作り込み
- フレンド機能
- チャット
- SNS共有
- 全体ランキング
- 高度な不正検知
- デスクトップ常駐化

## 基本思想

このプロジェクトは、作業時間の長さそのものを称賛するものではない。
45〜60分前後の集中と、5〜10分前後の休憩を推奨し、
120分を超える長時間連続作業は効率面で不利になるように設計する。

つまりこれは、
「時間記録アプリ」ではなく、
「集中の型を育てるアプリ」である。

## 技術方針

- Node.js / TypeScript 前提
- Web版を先に作る
- 将来的なアプリ化を想定し、時刻ベースでセッション状態を持つ
- ロジックをUIに直書きしない
- 計算ロジックは再利用可能な関数として切り出す

## 開発の進め方

1. 仕様を読む
2. タスクを小さく切る
3. 1本の体験を縦に通す
4. 差分を確認する
5. 小さい単位でコミットする

## まず最初に通すべき流れ

- 休憩
- 再開
- 終了
- 結果表示
- スコア計算
- 経験値加算
